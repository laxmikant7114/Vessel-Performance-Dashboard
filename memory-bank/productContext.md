# Product Context: Gekko Vessel Analytics Dashboard

## Why This Project Exists
The maritime industry requires real-time visibility into vessel performance and supplier relationships to optimize operations and reduce costs. This dashboard addresses the need for centralized, accessible business intelligence for fleet management.

## Problems It Solves
1. **Data Fragmentation**: Consolidates vessel performance data from multiple sources
2. **Accessibility**: Makes complex analytics accessible to non-technical stakeholders
3. **Decision Support**: Provides visual insights for operational and strategic decisions
4. **Supplier Management**: Tracks supplier performance metrics for better vendor relationships

## How It Should Work
### User Experience Flow
1. **Landing**: User arrives at professional dashboard with Gekko branding
2. **Navigation**: Sidebar provides clear vessel/supplier selection
3. **Analysis**: Interactive Power BI reports load seamlessly in main content area
4. **Insights**: Users can drill down into specific metrics and timeframes

### Key Features
- **Vessel-Specific Analytics**: Separate dashboards for Java and Deep Runner vessels
- **Supplier Performance**: Dedicated Harvey Supplier analysis
- **Interactive Reports**: Full Power BI functionality embedded
- **Responsive Design**: Works across desktop and tablet devices

## Target Users
- **Fleet Managers**: Need operational oversight and performance metrics
- **Operations Teams**: Require detailed vessel performance data
- **Procurement Teams**: Use supplier performance data for vendor decisions
- **Executives**: Need high-level insights for strategic planning

## Success Metrics
- Easy navigation between different analytical views
- Fast loading of embedded Power BI reports
- Professional presentation that builds stakeholder confidence
- Actionable insights leading to operational improvements

## User Experience Goals
- **Simplicity**: One-click access to different analytical views
- **Professionalism**: Clean, corporate design that inspires confidence
- **Performance**: Fast loading and responsive interactions
- **Clarity**: Clear data presentation without overwhelming users
